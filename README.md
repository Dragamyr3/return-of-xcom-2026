RETURN OF XCOM: LWotC Edition (2026)

A pre-planned collection for return to XCOM after some years... build on Long War of the Chosen.

I've seen many people asking how can someone run a setup of 500+ mods smoothly, wondering about some goodies out there, or just looking for essentials... well, as I was already building a collection from scraps of my previous one, thought why not share it.

At the moment I haven't started my ultimate playthrough yet. Testing & updating the list, hoping to fully optimize it by the end of MAY.

☬ Everything should be fully compatible/up to date.
☬ Needs further testing, yet so far zero errors, crashes or random freezing.
☬ Have worked on this for around a month, still have a couple of things to adjust.
☬ Feel free to check out my collection for some cool mods, or use it as a base for your playthrough.
☬ Rate this, favorite & comment, if you find any issues.

✦ [WOTC] Invisible Parts For Hero Units
✦ [WOTC] Reliable Ever Vigilant
✦ Primary Secondaries

IGNORE IT❗ or go to "missing dependencies" panel & checkmark 'ignore'.
This collection has improved versions: True Primary Secondaries [BETA]), [WOTC] Invisible Body Parts For All (v1.3) & LW2 Perks, [WOTC] Reliable Ever Vigilant Redux + Secondary Weapons are already part of base LWotC, you only need them while using basic WotC DLC version.

⚠️ There are some minor inconsistencies with 'Viper's Classes', but it shouldn't cause any game breaking issues & can be fixed by some basic editing: "problem with all proficiency classes that have GTS. Buying the upgrade won't unlock that ability on that specific class. The reason because you forgot to add AllowedClasses[0] = ClassName in the XComGtsEditor.ini."
